Projects

Project 1: My Portfolio Website Project

The current website is a personal project I created to showcase my work and skills. It includes an overview of my background, selected projects, and contact information. It is still under construction and I will be adding more features soon.

Key Features:

  • Built with HTML, CSS, JavaScript for the frontend.
  • Lambda, DynamoDB, and S3 for the backend.
  • Certificate Manager for HTTPS support.
  • Hosted on AWS with CloudFront for fast global delivery.
  • Route53 for domain management.
  • IAM for secure access control.
  • Fully responsive layout.
